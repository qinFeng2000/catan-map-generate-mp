# 5–6 人数字牌配置变更设计

## 目标

使 5–6 人扩充版数字牌配置与当前规则文档一致：数字 `2` 和 `12` 各 2 个，数字 `3`、`4`、`5`、`6`、`8`、`9`、`10`、`11` 各 3 个，总数保持 28 个。

## 变更范围

- 修改 `BOARD_PRESETS.extension.numberBag`：`2` 从 1 个增加到 2 个，`12` 从 3 个减少到 2 个。
- 保持基础版配置、资源配置、生成规则、搜索预算、页面和渲染逻辑不变。
- 在预设测试中增加扩充版数字频次的精确断言，避免只校验总数而遗漏配比错误。

## 实现与验证

先添加会在旧配置下失败的回归测试，再修改扩充版预设使其通过。运行预设测试，并运行受数字牌配置影响的数字搜索和生成测试；最后执行项目现有的完整测试或等价校验。

## 验收标准

- 扩充版 `numberBag` 总数为 28。
- `2` 和 `12` 各出现 2 次。
- 其余有效数字各出现 3 次。
- 基础版数字配置不变。
- 数字搜索与整盘生成仍能在现有预算内成功。
